# Health Nutrition Agent (Google ADK)

This is an AI Health Agent built using **Google ADK* and **Gemini models**.  
It mainly focuses on **health and nutrition** and provides **diet suggestions based on the user’s health problems**.

---

#  What this Agent Does

- Acts as a **Nutrition Advisor**
- Gives **diet suggestions** based on problems like:
  - Fever  
  - Acidity  
  - Diabetes  
  - Low iron  
  - Digestion issues  
  - And many more…
- Uses **Google Search Tool** (via ADK) to get updated information
- Gives **structured answers**  
- Maintains **session memory**
- Supports **ADK Web UI**
- we have used **gemini-2.5-flash-lite**
